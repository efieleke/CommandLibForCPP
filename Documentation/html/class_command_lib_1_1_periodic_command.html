Detailed Description

Represents a Command that repeats periodically at a specified interval

If more dynamic control is needed around the period of time between executions, use RecurringCommand instead.

enum CommandLib::PeriodicCommand::IntervalType
strong

Defines how the interval between command executions is performed within a PeriodicCommand

Enumerator
PauseBefore 

Pause first, then run the command, and repeat as many times as specified

PauseAfter 

Run the command first, then pause, and repeat as many times as specified. There is no pause after the last execution of the command.

void CommandLib::PeriodicCommand::SkipCurrentWait ( )

If currently in the interval between command executions, skips the wait and executes the command right away. This only skips the current wait. It will not skip subsequent waits.

void CommandLib::PeriodicCommand::Stop ( )

Causes the command to stop repeating. This will not cause the command to be aborted. If the command to run is currently executing when this is called, it will be allowed to finish.

This is a no-op if this PeriodicCommand instance is not currently executing

std::atomic_uint CommandLib::PeriodicCommand::m_repeatCount

The number of times the command to run will execute. If the command to run is currently executing when this is changed, it will be allowed to finish, even if the repeat count is set to a number lower than the number of times already executed.

It is safe to change this member while this command is executing
