Detailed Description

This is a callback interface for Command starting and finishing events. Its intended use is for logging and diagnostics.

CommandTracer and CommandLogger are available implementations. You may add a monitor via the static Command::sm_monitors member of Command. Monitors added to that member will be called for every Command object that executes. To receive callbacks only for commands dispatched via a CommandDispatcher, use CommandDispatcher::AddMonitor(CommandMonitor*).
