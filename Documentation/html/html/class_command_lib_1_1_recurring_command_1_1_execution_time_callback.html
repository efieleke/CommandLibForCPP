virtual bool CommandLib::RecurringCommand::ExecutionTimeCallback::GetFirstExecutionTime ( std::chrono::time_point< std::chrono::system_clock > *  time)
pure virtual

Called when a <see cref"RecurringCommand"/> needs to know the first time to execute its underlying command to run.

Parameters
timeImplementations should set this to the first time to execute. If a time in the past is specified, the command to run will execute immediately. However, if this method returns false, the value set here will be ignored.
Returns
Implementations should return true to indicate that the command to run should be executed at the provided time. Returning false causes the RecurringCommand to finish execution.
